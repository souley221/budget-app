Enable Banking non configuré

Pour activer la synchronisation bancaire, créez un compte gratuit sur Enable Banking (PSD2 — BNP, SG, LCL, CA, Revolut…) :

  1. Inscrivez-vous sur enablebanking.com/sign-in (gratuit, sans carte)
  2. Allez dans Applications → New application
  3. Copiez votre Application ID et téléchargez la clé privée (.pem)
  4. Placez le fichier .pem à la racine du projet et créez le fichier .env :
EB_APPLICATION_ID=votre_application_id
EB_PRIVATE_KEY_PATH=./enablebanking-private.pem
APP_URL=http://localhost:3000

Redémarrez le serveur (npm start) après avoir créé le fichier.
